# 🧵 NeedlOS

**NeedlOS** es un sistema web desarrollado para la administración integral de una sastrería. El proyecto permite gestionar múltiples procesos internos del negocio, como el manejo de órdenes, inventario, usuarios, nómina, arqueo de caja, informes, inicio de sesión seguro, control de gastos, y mucho más.

---

## 🚀 Características Principales

- Inicio de sesión seguro con control de tipo de usuario (administrador y sastre).
- Gestión de órdenes y estado de pedidos.
- Control y seguimiento del inventario.
- Administración de usuarios y roles.
- Gestión de nómina y arqueos de caja.
- Reportes e informes detallados.
- Control y registro de gastos.
- Interfaz web moderna y accesible desde cualquier navegador actualizado.

---

## 🛠️ Tecnologías Utilizadas

- **Frontend:** React + Vite
- **Backend:** Java + Spring Boot
- **Base de Datos:** PostgreSQL
- **Contenedores:** Docker

---

## 🌐 Acceso y Uso

El sistema está diseñado para ser accesible a través de un navegador web moderno. Solo se requiere:

- Conexión a internet
- Navegador actualizado (Chrome, Firefox, Edge, etc.)

---

## 🧱 Estructura del Proyecto

El repositorio está organizado principalmente en dos ramas:

- `style`: Contiene el desarrollo del frontend (React + Vite).
- `backend`: Contiene la lógica del servidor y conexiones con la base de datos (Spring Boot + PostgreSQL).
